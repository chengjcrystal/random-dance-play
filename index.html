browse_catalog.exe_◻✕

Tap Add on any song and it drops straight into your library with a clip already set. No links to paste. Or load a whole playlist in one tap.

add_songs.exe_◻✕

Not in the catalog? Paste any YouTube links here, one per line. Names fill in automatically and each gets a starter clip. Optional: put a name before the link.

how_to_run.txt_◻✕

Serve over http(s). From this folder run python3 -m http.server 8000 then open http://localhost:8000. Works on any static host too. To play on a TV, open it in a desktop browser and cast the tab. Your library is saved in this browser, so you only build it once.

Set Clip

Scrub the video, then tap Set Start and Set End where you want the dance section. Preview to confirm.
